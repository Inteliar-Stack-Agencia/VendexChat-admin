# Vendexchat Admin

Merchant dashboard for Vendexchat.
Manage products, orders, pricing and store configuration.

## Stack
- Next.js
- TypeScript
- Tailwind
- Cloudflare Pages
Objetivo: panel admin para tiendas VendexChat

Stack: Next.js + Tailwind + Cloudflare Pages

Deploy: Cloudflare Pages conectado a este repo

Env vars: NEXT_PUBLIC_API_BASE, NEXT_PUBLIC_TENANT_SLUG
